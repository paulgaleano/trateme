4 Objetivos

OBJ-0001 Objetivo general
Versión 1.0 ( 22/08/2018 )
Autores
  • Andrés López
  • Jean Paul Galeano
  • Jhon Medina
  • Richard Ruiz
Fuentes
  • Jean Paul Galeano
Descripción El sistema deberá Construir una línea de productos de sofware en el dominio del transporte terrestre de mercancía.
Subobjetivos
  • [OBJ-0002] Programar recogida: El sistema deberá Construir una línea de productos que permita gestionar la información de las programaciones de recogida de mercancía.
  • [OBJ-0003] Recibir mercancía: El sistema deberá Construir una línea de productos que permita gestionar la información de los recibos de mercancía.
  • [OBJ-0004] Transportar mercancía: El sistema deberá Construir una línea de productos que permita gestionar la información del transporte de mercancía desde el orígen hasta el destino.
  • [OBJ-0005] Entregar mercancía: El sistema deberá Construir un línea de productos que permita gestionar la información de las entregas de mercancía.
  • [OBJ-0006] Rastrear: El sistema deberá Construir un línea de productos que permita rastrear interna y externamente el proceso en el que se encuentra un envío de mercancía.
Importancia vital
Urgencia inmediatamente
Estado en construcción
Estabilidad alta
Comentarios Ninguno

Descripción El sistema deberá almacenar la información correspondiente a la programación del servicio de recogida. En concreto:
Datos específicos

  • Identificación remitente
  • Nombre remitente
  • Dirección remitente
  • Código DANE remitente ( Ciudad asociada a la dirección de recogida.)
  • Teléfonos remitente
  • Identificación destinatario
  • Nombre destinatario
  • Dirección destinatario
  • Código DANE destinatario ( Ciudad asociada a la dirección de entrega.)
  • Teléfonos destinatario
  • Token ( Información de autenticación.)
